re# Spy-X-Family-MachineProject
CCPROG1 - Machine Project



<h1>Prerequisites:</h1>

> GCC
> ncurses



<h1>How to run?</h1>

  > Windows
> 
  ```gcc -Wall -std=c99 main.c -o main.exe -lncurses -DNCURSES_STATIC```
  
  ```main.exe```

  > Linux
> 
  ```gcc -Wall -std=c99 main.c -o main.exe -lncurses```

  ```./main.exe```
